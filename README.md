# WPNAS Kit

Browse and install plugins from WPNAS.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm start
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Structure

- `src/`: React source code.
- `build/`: Compiled assets (generated).
- `wpnas-kit.php`: Main PHP file.
